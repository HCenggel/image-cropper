✨ 核心特性

图片变换

支持拖动平移、滚轮缩放（0.1x - 10x）、旋转、水平/垂直翻转等操作

裁剪模式

自由模式（可调整大小）和固定模式（固定尺寸），满足不同需求

桌面优化

专门针对桌面端鼠标操作优化，提供流畅的交互体验

灵活导出

支持JPEG和PNG格式导出，质量可调（0.1 - 1.0），返回Base64数据

纯JavaScript

无需任何框架依赖，轻量级代码，快速集成到任何项目

Canvas渲染

基于Canvas API绘制，最小化CSS依赖，性能优异

🚀 快速开始

JavaScript 
// 创建裁剪器实例
const cropper = new ImageCropper('#container', {
  width: 500,
  height: 500,
  cropMode: 'free'
});

// 加载图片
await cropper.loadImage('path/to/image.jpg');

// 导出裁剪后的图片
const base64Data = cropper.export('image/jpeg', 0.9);
console.log(base64Data);
HTML 
<!-- 引入库文件 -->
<script src="image-cropper.js"></script>

<!-- 创建容器 -->
<div id="container"></div>

void

flipVertical()

void

setScale(scale)

void

setCropMode(mode)

设置裁剪模式

void

export(format, quality)

string

reset()

void
